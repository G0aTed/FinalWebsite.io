Access the Internet



On every device, in order to access the internet, an essential piece of software is the web browser, which allows for the device to recieve and send information from the internet. It has an address bar, which when a URL (web address) is inputted to, sends said URL to the Domain Name System servers (DNS) which then sends back the corresponding IP address of the website. Once it recieves the HTML file of the page it is attempting to access, the web browser can then display it for the user to see.

Each device also has its own unique IP address, which allows for the servers to assign a destination for the files the device is requesting.




A router is used to send data to a specific destination on a network, a home router, also known as a hub, will route packets from one part of a network to another.

A router, asides from routeing packets, can also:
- Act as a wireless access point for Wi-Fi connections
- Connect the local network to the internet using a modem
- Assigns IP addresses to each device that is connected to it

A DHCP server is built into home routers, which when a new device is connected to it, assigns it a new IP address.




A modem, also known as a modulator-demodulator, is a device that converts data from a digital format into a format suitable to be transmitted via a analog medium. Examples of which can be telephone or radio.

It connects a home router to the Internet Service Provider that allows for the access to the Internet. The connection is made via a cable (for cable or fiber internet) or phone line (DSL) from outside your house that plugs into the back of your modem. Your modem shares this connection with a computer or a router via an Ethernet cable.




The Internet is a vast network that connects computers all over the world. Through the Internet, people can share information and communicate from anywhere with an Internet connection.

The Internet provides a capability so powerful and general that it can be used for almost any purpose that depends on information, and it is accessible by every individual who connects to one of its constituent networks.

It is here that data can flow accross the world virtually instantaneously, and have completely revolutionalized the lifestyles of modern day humans.

Fun fact: The entirety of the internet, weighed by its electrons, comes in at only 50 grams, the weight of a large strawberry.
